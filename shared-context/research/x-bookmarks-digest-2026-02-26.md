3. **Claude Code's "App Store" (ClawHub) now has 250K+ agent skills** — @dr_cintas frames this as a landmark shift: skills are becoming a standard unit of AI capability packaging. Works with Claude Code, VS Code, and Antigravity. Free. This is the infrastructure Ridley Research's consulting product lives on.

4. **Claude-Flow is the #1 agent framework on GitHub** — @heygurisingh reports it can run 60+ agents simultaneously with cross-agent learning. One plans, one codes, one tests, one reviews security. This is the leading architecture pattern for autonomous agent orchestration.

5. **@Akashi203 open-sourced an agent OS in Rust (OpenFang)** — 137K lines, MIT licensed, inspired by OpenClaw. Agents run in WASM sandboxes like OS processes. This is a serious alternative kernel-level approach worth tracking as competition to OpenClaw-native deployments.

6. **Blackbox released "Claudex Mode"** — Claude Code implements, Codex verifies and tests. Dual-agent coding pattern going mainstream. Not just experimental; shipping in a named product.

7. **@svpino documented a Claude skill for pulling structured web data** — returns tables and structured data instead of raw text. Immediately useful for Berean's research pipeline.
